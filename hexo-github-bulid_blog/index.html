写作步骤

  • git pull 拉取最新的 hexo 分支
  • hexo new post 'title' 新建文章
  • hexo clean 清理之前的静态站文件
  • hexo g 重新生成静态站文件
  • git add .
  • git commit -m "message"
  • git push 推送到 hexo 分支
  • hexo d 推送到 master 分支
